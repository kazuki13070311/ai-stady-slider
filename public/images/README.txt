# 画像ファイルの準備方法

スライドで使用する画像ファイルは以下のように準備してください：

1. 提示されていたSlackのお問い合わせチャンネルの画像を「image1.png」としてこのフォルダに保存
2. BANリスト管理画面の画像を「image.png」としてこのフォルダに保存

## 画像の配置場所

このフォルダ（/public/images/）に画像を配置すると、スライド内で以下のパスで参照されます：
- `/images/image1.png`
- `/images/image.png`

## 注意事項

- 画像ファイルの名前は必ず上記の名前にしてください
- 画像のサイズが大きい場合は、適切なサイズに圧縮することを推奨します
- スライドはすでに画像を表示するように設定されています
